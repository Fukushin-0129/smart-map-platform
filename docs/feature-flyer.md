# Feature: チラシ配布

## 目的

チラシ配布先を地図上で管理し、担当者、配布状況、配布日、配布枚数、メモ、写真、履歴を現地で素早く更新できるようにします。

## MVP 項目

- 配布状況4種。
- 担当者。
- 配布日。
- 配布枚数。
- メモ。
- CSV インポート。
- CSV エクスポート。
- ルート候補。
- レイヤー表示切替。

## 配布状況

- 未配布
- 配布済み
- 配布不可
- 不在

## Place customFields 例

```json
{
  "propertyName": "サンプルマンション",
  "units": 30,
  "distributionStatus": "未配布",
  "distributionDate": "2026-01-10",
  "distributedCount": 30,
  "area": "吹田",
  "propertyType": "マンション",
  "schoolDistrict": "第一中学校区"
}
```

## AI 要件

- 2〜10人に分割。
- 徒歩、自転車、自動車で最適化。
- 担当ごとの件数・距離・時間を均等化。
- Google Maps へ受け渡し。

## 画面要件

- 現地活動モードでは地図をほぼ全画面にする。
- マーカータップ後、配布済み・不在・配布不可をワンタップで更新する。
- 必要時のみ詳細入力を表示する。
- 管理画面で CSV インポートと担当者設定を行う。
