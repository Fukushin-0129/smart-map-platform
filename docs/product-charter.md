# Product Charter

## プロダクト名

Smart Map Platform

## 最上位コンセプト

- 現地で得た知識を、地図に残し、仲間と育てる。
- 場所に知識を重ねる。

## 基本思想

Smart Map Platform は、特定用途専用の地図アプリではありません。サラダ店、チラシ配布先、ダンススタジオ、営業先、不動産、観光地、防災施設、公共施設など、あらゆる「場所」に現地で得た情報を蓄積・共有・活用できる汎用ユーティリティアプリです。

## 解決する課題

- 現地で得た情報が個人の記憶、チャット、紙、表計算ファイルに分散する。
- 地図上の位置情報と、現場メモ・写真・履歴・担当者・状態が分離している。
- 小規模チームが使いやすい地図共有ツールは、用途が限定されるか設定が重い。
- 営業、配布、点検、観光、コミュニティ活動で「次にどこへ行くべきか」が分かりにくい。

## 中心概念

- Project / Channel: テーマ、メンバー、権限、表示項目を分ける単位。
- Place: すべての地点を表す共通データ。
- Knowledge: 場所に紐づく知識、コメント、評価、写真、タグ、属性。
- History: 状態変更、訪問、配布、更新などの履歴。
- User: 利用者。
- Permission: オーナー、編集者、閲覧者などの権限。
- Layer: 表示切替、分類、用途別可視化の単位。
- AI: ルート、振り分け、要約、重複検出、候補提示を支援する機能群。

## 想定利用シーン

- 北摂サラダマップ
- 吹田チラシ配布2026
- 関西ダンススタジオ
- 営業訪問マップ
- 防災設備点検
- 観光スポット共有
- 物件管理マップ
- 地域コミュニティマップ

## 非目標

- 単一用途だけに最適化された閉じたアプリにはしない。
- 既存地図サービスの単純なコピーにはしない。
- 最初から大規模 GIS ツールを目指さない。
- 現場で素早く使えない重い入力フォームを中心にしない。

## 成功指標

- 初回ユーザーが10分以内にプロジェクトを作成し、Place を登録できる。
- 現地活動中に主要操作を3タップ以内で完了できる。
- CSV から100件以上の地点を取り込み、地図上で状態管理できる。
- 既存 localStorage データを失わずに次世代データモデルへ移行できる。
- チラシ配布とサラダマップの2用途で MVP として販売可能な価値を示せる。
