# Feature: サラダマップ

## 目的

サラダ店、カフェ、テイクアウト店などを地図上に登録し、写真、評価、コメント、行った日、行きたい、お気に入りを蓄積します。

## MVP 項目

- 店舗登録。
- Google Places 候補登録。
- 写真。
- コメント。
- 評価。
- レイヤー。
- 共有を想定したデータ構造。

## 項目

- 店舗名。
- サラダの種類。
- おすすめメニュー。
- 価格帯。
- 写真。
- 評価。
- 行った日。
- 行きたい。
- お気に入り。
- コメント。

## Place customFields 例

```json
{
  "saladTypes": ["チョップド", "ボウル"],
  "recommendedMenu": "グリルチキンサラダ",
  "priceRange": "1000-1500円",
  "visitedAt": "2026-04-01",
  "wantToGo": true,
  "favorite": false
}
```

## AI 要件

- 好みに近い店舗の提案。
- 似た雰囲気のカフェ提案。
- 現在地周辺の未登録候補。
- おすすめルート。

## 既存機能の活用

- 手入力店舗登録。
- Google Places 検索。
- 写真 GPS からの候補検索。
- KML レイヤー。
- localStorage 保存。
